##  Módulo Matemático
**Archivo:** `modulo_matematico.py`  
**Rama:** `modulo_matematico`  
**Descripción:** Realiza operaciones matemáticas entre dos números ingresados por el usuario.

### Operaciones disponibles:
-  Suma
-  Resta
-  Multiplicación
-  División
-  Potenciación
-  Raíz cuadrada

### Ejemplo de uso:
```
Introduce el primer número: 9
Introduce el segundo número: 3
La suma de 9.0 y 3.0 es: 12.0
La raíz cuadrada de 9.0 es: 3.0
```

---

##  Módulo Cuento
**Archivo:** `modulo_cuento.py`  
**Rama:** `modulo_cuento`  
**Descripción:** Cuento interactivo narrado en consola sobre un equipo de 6 compañeros
que enfrentan un reto inesperado usando sus habilidades únicas.

### Características:
- Narración con pausas dramáticas usando `time.sleep()`
- Presentación de cada personaje con sus habilidades
- Historia con inicio, desarrollo y desafío final

##  Módulo Conversión
**Archivo:** `modulo_conversion.py`  
**Rama:** `modulo_conversion`  
**Descripción:** Convierte entre distintas unidades de medida en tres categorías.

### Conversiones disponibles:

** Distancia**
- Kilómetros → Metros
- Centímetros → Metros

** Masa**
- Libras → Kilogramos
- Gramos → Kilogramos

** Temperatura**
- Kelvin → Celsius
- Fahrenheit → Celsius

---

##  ¿Cómo ejecutar?

1. Abre la carpeta del proyecto en **Visual Studio Code**
2. Abre el archivo del módulo que deseas ejecutar
3. Presiona el botón **▶ Run** (arriba a la derecha)
4. O abre la terminal en VS Code y escribe:
```
python modulo_matematico.py
python modulo_cuento.py
python modulo_conversion.py
```
